# SHIVSHANKAR BIOCOAL ENERGY Website

This is a professional, corporate-style website for SHIVSHANKAR BIOCOAL ENERGY, a biomass briquettes manufacturing unit and agri-waste management agency.

## Features

- Responsive design with modern corporate styling
- Clean, industrial look with dark green, charcoal grey, white, and gold accents
- Sections include Hero, About Us, Services, Why Choose Us, Industries Served, Sustainability Impact, Call to Action, and Contact
- Contact forms for general inquiries and bulk supply requests
- Smooth scrolling navigation
- Basic animations on scroll

## File Structure

- `index.html` - Main HTML file with all content
- `styles.css` - CSS styles for layout and design
- `script.js` - JavaScript for interactivity (smooth scrolling, form handling, animations)

## How to View

1. Open `index.html` in any modern web browser
2. The website is fully self-contained and doesn't require a server

## Customization

- Edit `index.html` to modify content
- Update `styles.css` to change colors, fonts, or layout
- Modify `script.js` for additional interactivity

## Browser Compatibility

Tested on modern browsers (Chrome, Firefox, Safari, Edge). May require updates for older browsers.
